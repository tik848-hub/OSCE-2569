Deploy เป็น Web App
กด Deploy → New deployment → เลือก type: Web App
Execute as: Me
Who has access: Anyone
กด Deploy → Copy URL → วางในช่องด้านบน

พร้อมใช้งาน
กรอกคะแนนครบ 8 ฐาน → มาที่แท็บ "ส่งผล" → กดส่ง → ข้อมูลเข้า Google Sheets อัตโนมัติ ข้อมูลจะถูกจัดเก็บพร้อม Timestamp ทันที
ต่อยอด: สร้าง Dashboard ด้วย Looker Studio
1. ไปที่ lookerstudio.google.com → Create → Report
2. เลือก Data Source: Google Sheets → เลือกไฟล์ OSCE
3. เพิ่ม Chart: Bar chart (คะแนนรายฐาน), Scorecard (% ผ่าน), Table (รายชื่อไม่ผ่าน)
4. เพิ่ม Filter: หน่วยงาน, วันที่
5. Share link ให้หัวหน้าฝ่ายดู Realtime ได้ทันที
